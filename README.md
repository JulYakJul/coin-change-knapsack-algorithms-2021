# Лабораторная работа по алгоритмам (Программная инженерия, УлГТУ)

[English version available here](./README_EN.md)

Этот проект содержит решения двух классических алгоритмических задач:  
1. **Задача о размене монет** (жадный алгоритм)  
2. **Задача о рюкзаке** (метод удельной стоимости)  

---

## 1. Размен монет (Жадный алгоритм)

**Логика задачи**:  
Необходимо разменять заданную сумму денег минимальным количеством монет, используя стандартные номиналы (например, 5000, 2000, 1000 руб. и т.д.).

**Реализация**:  
1. Пользователь вводит сумму (`sum`).
2. Алгоритм последовательно выбирает наибольший доступный номинал из массива `nominals`, делит сумму на этот номинал и вычитает полученное значение из общей суммы.
3. Процесс повторяется, пока сумма не станет нулевой.  

**Пример работы**:  
Введите сумму денег: 123
В сумме 2 монет номиналом 50 руб
В сумме 1 монет номиналом 20 руб
В сумме 1 монет номиналом 2 руб
В сумме 1 монет номиналом 1 руб

## 2. Задача о рюкзаке (Метод удельной стоимости)

**Логика задачи**:  
Необходимо выбрать набор предметов с максимальной суммарной стоимостью, не превышающий вместимость рюкзака (`C`).

**Реализация**:
1. Для каждого предмета рассчитывается "удельная стоимость" (стоимость/массу)
2. Предметы сортируются по убыванию этого коэффициента
3. Алгоритм добавляет предметы в рюкзак, пока их общий вес не превысит `C`

**Пример вывода**:
Подходящие предметы: 5 1  
Общий вес: 20  
Сумма: 48
